# week07 Lab
自訂錯誤條件
## Email
- 必填 (值經過 trim() 後不得為空字串)
- Email 格式 (使用瀏覽器內建的 checkValidity()) 
- 錯誤訊息:必須輸入有效的Email格式
## phone
- 必填 (值經過 trim() 後不得為空字串)
- 格式 (必須是精確的 10 位數字：/^[0-9]{10}$/)
- 錯誤訊息:電話號碼須為10碼
## password
- 必填 (值經過 trim() 後不得為空字串)
- 長度 (密碼長度必須 大於等於 8 碼)
- 錯誤訊息:密碼至少需要8碼
- 內容 (密碼必須 同時包含 至少一個英文字母 ([a-zA-Z]) 和一個數字 ([0-9]) )
- 錯誤訊息:密碼需包含英文與數字
## confirmPassword
- 必填 (值經過 trim() 後不得為空字串)
- 密碼比對 (輸入值必須與 password 欄位的值一致)
- 錯誤訊息:兩次輸入的密碼不一致
## terms (服務條款)
- 必填 (值經過 trim() 後不得為空字串)
- 必選 (必須被 勾選 (input.checked 為 true))
- 錯誤訊息:請勾選服務條款
## interestTags 驗證規則
- 必須至少選擇一個興趣
- 錯誤訊息:請至少選擇一個興趣
## 加分項目
- 實作「重設」按鈕，清除欄位與錯誤訊息
